# Plano-de-Capacita-o-T-cnica
Este repositório documenta meu Plano de Capacitação Técnica em React. O foco é a transição de arquiteturas PHP para SPAs modernas, com entregas semanais sobre JS Moderno, Hooks, Integração com APIs e Atomic Design. O objetivo é alcançar autonomia técnica e padronização no desenvolvimento de software.
